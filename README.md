# gigigo-encryptation-util
### Library to encrypt information with the following algorithms
 * In Java library:
  * Md5
  * Sha256
  * Aes256
 
 * In Android library:
  * Base64
  * An algorithm to encrypt and decrypt data: ![GitHub Logo](/screen/EncryptionImageAlgorithm.png)

To use this library:
```gradle
compile com.github.Gigigo-Android-Devs.gigigo-encryptation-util:encryptation-java-library:1.0
compile com.github.Gigigo-Android-Devs.gigigo-encryptation-util:encryptation-android-library:1.0
```
